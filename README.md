# 🎮 消消乐游戏

一个简单的网页版消消乐游戏，使用纯 HTML、CSS 和 JavaScript 开发。

## 🎯 游戏规则

1. 点击一个方块，再点击相邻的方块进行交换
2. 当3个或以上相同颜色的方块连成一线时，它们会被消除
3. 消除方块可以获得分数（每个方块10分）
4. 方块消除后，上方的方块会下落填充空位
5. 新的方块会从顶部生成

## 🚀 如何运行

### 方法1：直接打开
直接用浏览器打开 `index.html` 文件即可开始游戏。

### 方法2：本地服务器
```bash
# 使用 Python 启动本地服务器
python3 -m http.server 8000

# 然后在浏览器访问
# http://localhost:8000
```

## 🔧 代码说明

### index.html
- 定义了游戏的基本结构
- 包含分数显示和重新开始按钮
- 游戏棋盘容器

### style.css
- 渐变背景
- 网格布局（8x8）
- 方块样式和悬停效果
- 选中状态高亮

### game.js
核心功能：
- `initGame()` - 初始化游戏
- `createBoard()` - 创建棋盘
- `renderBoard()` - 渲染界面
- `handleCellClick()` - 处理点击
- `swapCells()` - 交换方块
- `isPartOfMatch()` - 检测匹配
- `processMatches()` - 消除匹配
- `dropCells()` - 方块下落
- `fillEmpty()` - 填充空位
